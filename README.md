# LUMEN Shader Instrument

A live WebGL instrument for shaping procedural light, motion, matter, and color.

## Development

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

## Build

```bash
npm run build:pages
```

The GitHub Pages entry point lives in `github-pages/`. The full application source lives in `app/`.

Live site: https://nlevin.github.io/lumen-shader-instrument/
